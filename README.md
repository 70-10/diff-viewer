# Diff Viewer

A web application for visually comparing differences between two texts.

## Demo

https://70-10.github.io/diff-viewer/

## Features

- Real-time comparison of differences between two texts
- Shareable URL generation

## Usage

1. Access the demo site
2. Enter the texts you want to compare in the left and right text areas
3. Differences will be displayed automatically
4. To share, click the "Copy" button at the bottom to copy the URL
